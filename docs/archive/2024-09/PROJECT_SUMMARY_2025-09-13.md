# 2025-09-13 포트폴리오 & 블로그 사이트 개발 작업 요약

## 🎯 메인 작업 내용

### 1. 마크다운 지원 시스템 구현
**목적**: Work와 Archive 콘텐츠 작성 시 리치 텍스트 지원

#### 기술적 구현사항:
- **라이브러리 설치**: 
  - react-markdown (마크다운 파싱 및 렌더링)
  - react-syntax-highlighter (코드 블록 문법 강조)
  - @types/react-syntax-highlighter (TypeScript 타입 정의)

- **MarkdownEditor 컴포넌트 생성**:
  - 탭 기반 UI (편집/미리보기)
  - 실시간 마크다운 미리보기
  - 코드 블록에 다크 테마 문법 강조
  - 마크다운 사용법 도움말 제공
  - 커스텀 스타일링 (Tailwind CSS)

- **적용 범위**:
  - Work 작성/수정 폼의 설명 필드
  - Archive 작성/수정 폼의 내용 필드
  - Work 상세 페이지 (블루 테마)
  - Archive 상세 페이지 (퍼플 테마)

#### 마크다운 기능:
- 제목 (H1, H2, H3)
- 강조 (**굵게**, *기울임*)
- 링크 및 이미지
- 코드 블록 (문법 강조 지원)
- 목록 (순서 있음/없음)
- 인용문
- 표 지원
- 사용자 정의 스타일링

- Node.js 스크립트 파일의 require() 문제

#### ESLint 설정:
