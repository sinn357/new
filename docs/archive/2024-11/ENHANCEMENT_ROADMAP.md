# Blog Enhancement Roadmap

> **목적**: Work와 Archive 섹션을 극대화하는 단계별 업그레이드 계획
> **시작일**: 2025-11-18
> **전체 기간**: 3-4개월 예상

---

## 📋 전체 개요

총 15개 아이디어를 3단계(Phase A/B/C)로 분류하여 순차적으로 구현

- **Phase A (2주)**: 즉시 효과가 큰 기본 기능 4개
- **Phase B (1개월)**: 중급 기능 및 통합 5개
- **Phase C (2-3개월)**: 고급 기능 및 자동화 6개

---

## 🚀 Phase A: 즉시 효과 (2주)

### A-1. Archive 리뷰 전용 기능 ⭐
**목표**: 리뷰 글에 평점 및 스포일러 기능 추가

- [ ] **평점 시스템**
  - Archive 모델에 `rating` 필드 추가 (1-5)
  - 별점 입력 UI (작성 페이지)
  - 별점 표시 UI (목록/상세 페이지)
- [ ] **스포일러 블러**
  - `[spoiler]...[/spoiler]` 마크다운 문법 지원
  - 토글 버튼으로 스포일러 내용 보기/숨기기
  - 블러 효과 CSS

**예상 작업**: DB 마이그레이션 + 컴포넌트 2개 + 마크다운 파서 수정

---

### A-2. Work/Archive 이미지 라이트박스 🖼️
**목표**: 이미지 클릭 시 확대 뷰 제공

- [ ] react-image-lightbox 또는 yet-another-react-lightbox 설치
- [ ] Work 상세 페이지에 적용
- [ ] Archive 상세 페이지에 적용
- [ ] 갤러리 모드 (여러 이미지 슬라이드)

**예상 작업**: 라이브러리 설치 + 이미지 컴포넌트 래핑

---

### A-3. Work Featured 프로젝트 🏷️
**목표**: 대표 작업물을 홈/Work 페이지에 강조 표시

- [ ] Work 모델에 `isFeatured` 필드 추가 (boolean)
- [ ] 관리자 페이지에서 Featured 토글 기능
- [ ] 홈 페이지에 "Featured Projects" 섹션
- [ ] Work 페이지 상단에 Featured 우선 표시

**예상 작업**: DB 마이그레이션 + UI 컴포넌트 수정

---

### A-4. 소셜 & 공유 기능 📱
**목표**: SNS 공유 및 Open Graph 최적화

- [ ] **Open Graph 메타태그**
  - Work/Archive 상세 페이지에 동적 OG 태그
  - 썸네일 이미지 자동 선택
- [ ] **공유 버튼**
  - Twitter, Facebook, LinkedIn 공유 컴포넌트
  - 클립보드 복사 기능
- [ ] **프린트 최적화**
  - @media print CSS 추가

**예상 작업**: 메타태그 생성 함수 + ShareButton 컴포넌트

---

## 🌟 Phase B: 중기 기능 (1개월)

### B-1. Work 케이스 스터디 포맷 📊
**목표**: 프로젝트 상세 페이지를 전문적인 케이스 스터디 형식으로 개편

- [ ] Work 모델 확장
  - `problem`, `solution`, `result` 필드 추가
  - `highlights` (JSON) - 주요 기능별 설명
  - `timeline` (JSON) - 작업 단계별 설명
- [ ] 상세 페이지 레이아웃 재설계
  - Problem → Solution → Result 섹션
  - 작업 과정 타임라인 시각화
  - 기술적 하이라이트 섹션
- [ ] Before/After 비교 슬라이더 (react-compare-image)

**예상 작업**: DB 마이그레이션 + 페이지 전체 리뉴얼

---

### B-2. Archive 여행 지도 통합 🗺️
**목표**: 여행 리뷰에 방문 장소 지도 표시

- [ ] Archive 모델에 `locations` 필드 추가 (JSON)
  - 위도/경도, 장소명, 설명
- [ ] Mapbox 또는 Google Maps 통합
- [ ] 여행 카테고리 필터링
- [ ] 지도 핀 클릭 시 해당 글 연결

**예상 작업**: 지도 라이브러리 + 데이터 모델 + 컴포넌트

---

### B-3. 통합 검색 & 발견 시스템 🔍
**목표**: Work + Archive 전체를 검색하고 태그 기반 탐색

- [ ] 전역 검색바 (헤더)
- [ ] PostgreSQL Full-text Search 또는 Algolia
- [ ] 검색 결과 페이지 (Work/Archive 통합)
- [ ] 태그 클릭 시 관련 글 모아보기
- [ ] "관련 콘텐츠" 추천 (같은 태그/카테고리)

**예상 작업**: 검색 API + UI 컴포넌트 + 추천 로직

---

### B-4. 연간 회고 페이지 📅
**목표**: 한 해 동안의 작업물과 경험 총정리

- [ ] `/year/2024` 페이지 생성
- [ ] 통계 대시보드
  - 프로젝트 수, 리뷰 수
  - 카테고리별 분포 차트
  - 월별 활동량 그래프
- [ ] Best of the Year 선정
- [ ] 타임라인 뷰

**예상 작업**: 통계 API + 차트 라이브러리 + 페이지

---

### B-5. Work 필터링 & 발견성 강화 🎯
**목표**: 프로젝트를 다양한 기준으로 필터링

- [ ] 기술스택 필터 (React, Node.js 등)
- [ ] 카테고리 탭 (Product/Media/Photography)
- [ ] 상태별 필터 (Completed/In Progress/Planned)
- [ ] 정렬 옵션 (최신순/인기순/기간순)
- [ ] URL 쿼리 파라미터 기반

**예상 작업**: 필터 UI + API 수정 + URL 상태 관리

---

## 🔥 Phase C: 장기 고급 기능 (2-3개월)

### C-1. Archive 메타데이터 자동화 🤖
**목표**: 외부 API로 영화/책/음악 정보 자동 입력

- [ ] TMDb API 연동 (영화)
  - 포스터, 감독, 배우, 개봉년도
- [ ] Google Books API (책)
  - 표지, 저자, 출판사, 페이지수
- [ ] Spotify API (음악)
  - 앨범 커버, 아티스트, 발매일
- [ ] Archive 작성 시 자동 검색 UI
- [ ] 메타데이터 필드 확장

**예상 작업**: API 통합 3개 + 검색 UI + DB 스키마

---

### C-2. Work 인터랙티브 쇼케이스 🎨
**목표**: 3D 효과와 인터랙티브 요소로 작업물 강조

- [ ] react-tilt로 3D 틸트 카드
- [ ] 호버 시 GIF/비디오 미리보기
- [ ] Parallax 스크롤링 (react-scroll-parallax)
- [ ] 풀스크린 갤러리 모드 (Swiper)

**예상 작업**: 여러 라이브러리 통합 + 애니메이션

---

### C-3. Archive 시각적 타임라인 📈
**목표**: 연대기순 리뷰 시각화

- [ ] 타임라인 뷰 페이지
- [ ] 월별 아카이브 네비게이션
- [ ] 카테고리별 필터 (영화/책/음악/여행)
- [ ] 태그 클라우드 시각화

**예상 작업**: 타임라인 컴포넌트 + 데이터 집계

---

### C-4. Archive 리뷰 템플릿 시스템 📝
**목표**: 카테고리별 맞춤 입력 폼

- [ ] 영화 리뷰 템플릿 (줄거리/연출/연기/총평)
- [ ] 책 리뷰 템플릿 (핵심 메시지/인상깊은 구절)
- [ ] 음악 리뷰 템플릿 (분위기/하이라이트 트랙)
- [ ] 여행 리뷰 템플릿 (방문 장소/맛집/팁)
- [ ] 템플릿 선택 UI

**예상 작업**: 폼 컴포넌트 4개 + 조건부 렌더링

---

### C-5. Archive 콘텐츠 풍부화 💎
**목표**: 리뷰 글을 더 풍부하게 표현

- [ ] 인용구 하이라이트 컴포넌트
- [ ] 이미지 컬렉션 그리드 (Pinterest 스타일)
- [ ] 감상 무드 태그 (😂웃긴/😭감동/🤔생각할거리)
- [ ] 관련 링크 섹션 (공식 사이트, 예고편)
- [ ] 개인 메모 필드

**예상 작업**: UI 컴포넌트 여러 개 + DB 필드

---

### C-6. 개인 브랜딩 강화 🌟
**목표**: 프로필과 연락 기능 업그레이드

- [ ] About 페이지 리뉴얼
  - 기술스택 비주얼
  - 성장 타임라인
  - 연락처 통합
- [ ] 뉴스레터 구독 (EmailJS)
- [ ] 소셜 미디어 Follow 버튼
- [ ] 포트폴리오 PDF 다운로드

**예상 작업**: About 페이지 + 뉴스레터 폼 + PDF 생성
